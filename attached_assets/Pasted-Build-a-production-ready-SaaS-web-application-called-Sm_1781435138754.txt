Build a production-ready SaaS web application called:

"Smart Tiffin & Student Analytics Platform"

The platform has TWO COMPLETELY INDEPENDENT USER TYPES:

1. Student
2. Tiffin Service Owner

Both user types can use the system independently.

A student does NOT need to join any tiffin service.

A tiffin owner can manage customers even if those customers do not have accounts.

However, there should be an OPTIONAL connection feature where students can join a tiffin service using an invite code.
